# WASD-to-arrows

A tiny Python script that remaps WASD keys to arrow keys on Windows.

## Requirements
### For .py
- Python 3
- [`keyboard`](https://pypi.org/project/keyboard/) library

```
pip install keyboard
```

### For .exe (Release)

## How to use

Run the exe,

Close the window to stop.

## What to use it for?

Any game that uses arrow keys but doesn't let you rebind them, ex.:

- **Undertale** / **Deltarune**
- Old Flash/browser games
- RPG Maker games
- Emulators
